# Template Cargon Index-04 (Standalone)

File `index-04-standalone.html` adalah template HTML standalone yang bisa dibuka langsung di browser tanpa perlu Flask atau server.

## Cara Menggunakan

1. **Buka file langsung di browser:**
   - Double-click file `index-04-standalone.html`
   - Atau buka dengan browser (Chrome, Firefox, Edge, dll)

2. **Melalui web server (disarankan):**
   - Jika menggunakan XAMPP, copy file ke folder `htdocs`
   - Buka melalui: `http://localhost/cargon/index-04-standalone.html`

## Struktur File

File ini berisi:
- ✅ Header dengan logo Cargon dan menu navigasi
- ✅ Hero section dengan teks "Transporting Your Business to New Heights Logistic"
- ✅ Background gelap dengan pola dotted
- ✅ Gambar truck di hero section
- ✅ Tombol "Track Shipment" di header
- ✅ Tombol "Our Service" di hero section
- ✅ About section
- ✅ Service section
- ✅ Footer

## Catatan

- Semua path CSS, JS, dan gambar menggunakan path relatif ke folder `apps/static/`
- Pastikan struktur folder tetap sama seperti struktur asli
- Jika ingin memindahkan file, pastikan path ke folder `apps/static/` masih benar

## Path yang Digunakan

- CSS: `apps/static/css/`
- JavaScript: `apps/static/js/`
- Images: `apps/static/img/`

## Jika Ingin Menggunakan di Folder Berbeda

Jika ingin memindahkan file ke folder lain, ubah semua path dari:
- `apps/static/` menjadi path yang sesuai dengan lokasi folder static Anda

Contoh jika file HTML ada di root dan static ada di `static/`:
- Ganti `apps/static/css/` menjadi `static/css/`
- Ganti `apps/static/js/` menjadi `static/js/`
- Ganti `apps/static/img/` menjadi `static/img/`
