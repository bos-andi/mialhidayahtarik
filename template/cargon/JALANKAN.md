# Cara Menjalankan Aplikasi Cargon

## Persyaratan
- Python 3.8 atau lebih baru
- pip (biasanya sudah termasuk dengan Python)

## Langkah-langkah

1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Jalankan Aplikasi**
   ```bash
   python run.py
   ```
   atau
   ```bash
   python -m flask run
   ```

3. **Buka di Browser**
   - Aplikasi akan berjalan di: http://127.0.0.1:5000 atau http://localhost:5000

## Catatan
- Pastikan Python sudah terinstall dan ada di PATH
- Jika menggunakan virtual environment, aktifkan terlebih dahulu
- Database SQLite akan dibuat otomatis saat pertama kali dijalankan
